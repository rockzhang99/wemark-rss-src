# 操作说明截图目录

把操作说明 HTML 中引用的截图放进本目录，文件名需与 `操作说明.html` 中一致，图片即可自动显示。

## 需要的截图清单

| 文件名 | 对应内容 |
|---|---|
| `step1-授权管理.png` | 授权管理页面（已授权/未授权状态） |
| `step1-扫码二维码.png` | 订阅管理页点「刷新授权」弹出的二维码 |
| `step2-添加公众号.png` | 添加公众号页面（文章链接/名称搜索入口） |
| `step3-订阅管理列表.png` | 订阅管理页（左侧列表+右侧文章表格+刷新/清理） |
| `step4-配置通知.png` | 配置信息页（notice 通知项输入框） |
| `step5-消息任务.png` | 消息任务页（添加任务：推送类型+cron） |
| `step6-生成RSS.png` | 订阅管理页「订阅」下拉（RSS/ATOM 等链接） |

未放入图片时，HTML 会自动显示虚线占位框，提示应放什么截图。
